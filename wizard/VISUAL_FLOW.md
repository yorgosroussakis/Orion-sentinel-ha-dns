## Color Scheme

- **Primary**: Blue (#2563eb) - Buttons, headings, links
- **Success**: Green (#16a34a) - Checkmarks, success messages
- **Warning**: Orange (#ea580c) - Warnings, paranoid profile badge
- **Background**: Light gray (#f8fafc)
- **Cards**: White (#ffffff)
- **Text**: Dark gray (#1e293b)
- **Text Light**: Medium gray (#64748b)

## Typography

- **Font**: System font stack (Apple System, Segoe UI, Roboto, etc.)
- **Headings**: Bold, larger sizes
- **Body**: 1rem (16px), 1.6 line height
- **Small text**: 0.875rem (14px) for hints and labels

## Interactive Elements

- **Buttons**: 
  - Primary: Blue background, white text
  - Secondary: Gray background, white text
  - Hover: Slight lift effect with shadow
  
- **Form inputs**:
  - 2px border, rounded corners
  - Focus: Blue border
  - Padding for easy touch/click

- **Radio cards**:
  - Clickable cards with subtle border
  - Hover: Blue border, slight background tint
  - Selected: Checkmark in top-right corner

- **Profile cards**:
  - Grid layout (3 columns on desktop, 1 on mobile)
  - Hover: Lift effect with shadow
  - Selected: Blue header background
